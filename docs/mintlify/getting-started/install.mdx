title: "Install"

---

tax-engine ships as a single precompiled binary — no runtime, no dependencies, no `npm install`.

## Download

<Tabs>
  <Tab title="Mac (Apple chip)">
    For M1, M2, M3, M4 Macs:

    ```bash
    curl -L https://github.com/filed-app/tax-engine/releases/latest/download/tax-macos-arm64 \
      -o /usr/local/bin/tax && chmod +x /usr/local/bin/tax
    ```

    Not sure which Mac you have? Apple menu → **About This Mac**. If it says "Apple M…" you want this one.
  </Tab>
  <Tab title="Mac (Intel)">
    For older Intel Macs:

    ```bash
    curl -L https://github.com/filed-app/tax-engine/releases/latest/download/tax-macos-x64 \
      -o /usr/local/bin/tax && chmod +x /usr/local/bin/tax
    ```
  </Tab>
  <Tab title="Windows">
    Download [`tax-windows-x64.exe`](https://github.com/filed-app/tax-engine/releases/latest/download/tax-windows-x64.exe) from the releases page.

    Then move it to a folder in your `PATH` and rename it to `tax.exe`.
  </Tab>
  <Tab title="Linux">
    ```bash
    curl -L https://github.com/filed-app/tax-engine/releases/latest/download/tax-linux-x64 \
      -o /usr/local/bin/tax && chmod +x /usr/local/bin/tax
    ```
  </Tab>
</Tabs>

## Verify the install

```bash
tax --help
```

You should see the available commands listed.

## Manual download

All binaries are available at [github.com/filed-app/tax-engine/releases](https://github.com/filed-app/tax-engine/releases).

| Platform | File |
|----------|------|
| Mac (Apple chip) | `tax-macos-arm64` |
| Mac (Intel) | `tax-macos-x64` |
| Windows | `tax-windows-x64.exe` |
| Linux | `tax-linux-x64` |

## Build from source

Requires [Deno](https://deno.land).

```bash
git clone https://github.com/filed-app/tax-engine
cd tax-engine
deno task compile
```

The binary lands in `dist/tax`.
