## Form 8949 — Capital Gains and Losses

```bash
tax form add --returnId ID --node_type f8949 '{ ... }'
```

One entry per lot sold. Add multiple `f8949` entries for multiple transactions.

| Field | Type | Description |
|-------|------|-------------|
| `description` | string | Description of property |
| `date_acquired` | string | Date acquired (MM/DD/YYYY or "Various") |
| `date_sold` | string | Date sold or disposed |
| `proceeds` | number | Gross proceeds |
| `cost_basis` | number | Cost or other basis |
| `adjustment_code` | string | IRS adjustment code (B, E, H, M, N, O, T, W) |
| `adjustment_amount` | number | Amount of adjustment |
| `reporting_category` | string | A, B, C, D, E, or F |

**Reporting categories:**

| Category | Description |
|----------|-------------|
| A | Short-term, covered (basis reported to IRS) |
| B | Short-term, noncovered |
| C | Short-term, other |
| D | Long-term, covered |
| E | Long-term, noncovered |
| F | Long-term, other |
