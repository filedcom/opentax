## 1098 Mortgage Interest

```bash
tax form add --returnId ID --node_type f1098 '{ ... }'
```

| Field | Type | Description |
|-------|------|-------------|
| `box1_mortgage_interest` | number | Mortgage interest received |
| `box2_outstanding_principal` | number | Outstanding mortgage principal |
| `box5_mortgage_insurance` | number | Mortgage insurance premiums |
| `box10_real_estate_taxes` | number | Real estate taxes paid from escrow |

## IRA Deduction Worksheet

```bash
tax form add --returnId ID --node_type ira_deduction_worksheet '{ ... }'
```

| Field | Type | Description |
|-------|------|-------------|
| `traditional_ira_contribution` | number | Traditional IRA contribution |
| `covered_by_retirement_plan` | boolean | Covered by workplace plan |
| `spouse_covered` | boolean | Spouse covered by plan |

The deduction phases out based on MAGI and whether you (or your spouse) are covered by a workplace retirement plan.
