## General — Taxpayer Info & Filing Status

```bash
tax form add --returnId ID --node_type general '{ ... }'
```

Required for every return. Sets identity, filing status, and dependents.

| Field | Type | Description |
|-------|------|-------------|
| `filing_status` | string | `single`, `married_filing_jointly`, `married_filing_separately`, `head_of_household`, `qualifying_surviving_spouse` |
| `dependents` | array | List of qualifying dependents |
| `blind` | boolean | Taxpayer is legally blind |
| `spouse_blind` | boolean | Spouse is legally blind |
| `age` | number | Taxpayer age (for standard deduction add-on) |
| `spouse_age` | number | Spouse age |

## Form 1040-ES — Estimated Tax Payments

```bash
tax form add --returnId ID --node_type f1040es '{ ... }'
```

| Field | Type | Description |
|-------|------|-------------|
| `q1_payment` | number | Q1 payment (due April 15) |
| `q2_payment` | number | Q2 payment (due June 15) |
| `q3_payment` | number | Q3 payment (due September 15) |
| `q4_payment` | number | Q4 payment (due January 15) |
