## W-2 Wage & Tax Statement

```bash
tax form add --returnId ID --node_type w2 '{ ... }'
```

| Field | Type | Description |
|-------|------|-------------|
| `box1_wages` | number | Wages, tips, other compensation |
| `box2_fed_withheld` | number | Federal income tax withheld |
| `box3_ss_wages` | number | Social Security wages |
| `box4_ss_withheld` | number | Social Security tax withheld |
| `box5_medicare_wages` | number | Medicare wages and tips |
| `box6_medicare_withheld` | number | Medicare tax withheld |
| `box8_allocated_tips` | number | Allocated tips |
| `box10_dependent_care` | number | Dependent care benefits |
| `box12_*` | various | Code A–FF boxes |
| `box13_statutory_employee` | boolean | Statutory employee checkbox |
| `box14_*` | various | Other compensation codes |
