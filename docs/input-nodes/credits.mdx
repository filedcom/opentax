## Form 2441 — Child and Dependent Care Credit

```bash
tax form add --returnId ID --node_type f2441 '{ ... }'
```

| Field | Type | Description |
|-------|------|-------------|
| `care_expenses` | number | Qualifying care expenses |
| `provider_ein` | string | Care provider EIN |
| `provider_name` | string | Care provider name |
| `qualifying_persons` | array | Dependents for whom care was provided |
| `employer_dependent_care` | number | Employer-paid dependent care (from W-2 box 10) |

## Form 8812 — Child Tax Credit / Additional Child Tax Credit

```bash
tax form add --returnId ID --node_type f8812 '{ ... }'
```

The engine computes the child tax credit automatically from dependents listed in the `general` node. Use `f8812` only for supplemental data such as Schedule 8812 inputs.

## Form 8863 — American Opportunity and Lifetime Learning Credits

```bash
tax form add --returnId ID --node_type f8863 '{ ... }'
```

| Field | Type | Description |
|-------|------|-------------|
| `student_name` | string | Student's name |
| `student_ssn` | string | Student's SSN |
| `aotc_expenses` | number | Qualified expenses for American Opportunity Credit |
| `llc_expenses` | number | Qualified expenses for Lifetime Learning Credit |
| `half_time_student` | boolean | At least half-time student |
| `first_four_years` | boolean | First four years of higher education |
